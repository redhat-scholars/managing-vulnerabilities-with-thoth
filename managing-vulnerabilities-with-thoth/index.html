Welcome to the Managing Vulnerabilities with Thoth Tutorial

Introduction

In this tutorial, we are going to demonstrate how to manage vulnerabilities in Python applications with Thoth (AIDevSecOps) via the Thamos CLI. This track includes an example of an application which uses Thoth’s recommendations to recommend a software stack for specific security requirements.

If you are interest in Thoth, feel free to follow @ThothStation Twitter handle or follow Thoth Station on YouTube.
